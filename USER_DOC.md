To start the project, the command is :
 
```bash
make
```


To stop it, if you want to keep data :
 
```bash
make clean
```


Else, the command is :
 
```bash
make fclean
```


To access the site, the url is "https://axbaudri.42.fr" and for the administration panel : "https://axbaudri.42.fr/wp-admin"
